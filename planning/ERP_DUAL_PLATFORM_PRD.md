# Phase 18: The "Autonomous Enterprise Ecosystem" Master PRD

## 1. Executive Summary
FutureBuild is executing a paradigm shift, transitioning from a localized project management tool into a dual-platform Agentic ERP/OS for residential construction. We operate under a strict Hub-and-Spoke Separation of Concerns:
- **FutureBuild OS (The System of Execution):** The core Business Management OS handling the deterministic CPM-res1.0 physics engine, corporate financials, workforce/HR, equipment tracking, and field execution UIs.
- **FB-Brain (The System of Connection):** The integration, analytics, and orchestration layer handling the No-Code Agent Workflow Builder, Maestro AI co-pilot, and all 3rd-party integrations (GableERP, LocalBlue, QuickBooks, municipal permit portals) via MCP and A2A protocols.

## 2. Core UI/UX Philosophy
- **Agent-as-Sieve:** Exception-based UI separating 'thoughts' from 'steps'—the agent proactively surfaces priorities so the user never has to "hunt" for data.
- **Why-Trails:** Explainable AI built into every autonomous decision. Users can click any action to see the deterministic math or reasoning behind it.
- **Spatially-Aware Reality Overlays:** Embedding data directly into physical context using BIM, Drone, and 360-degree photos.
- **Contextual Multimodality:** Voice-first chat for the field. Offline-capable voice memos that synchronize and process when connectivity returns.

## 3. The 3 Workspaces

1. **Portfolio Command Center (Executive View):**
   - **Role:** The Business OS for Executives.
   - **Aesthetic:** A sophisticated, high-altitude dark-mode dashboard with vibrant data visualizations.
   - **Layout/Features:** Focused on global risk heatmaps plotting out active projects, overlaid with real-time consolidated financial aggregates and portfolio-level compliance tracking.

2. **Agent Command Center (PM View):**
   - **Role:** The primary workspace for Project Managers.
   - **Aesthetic:** A premium "glassmorphism" dark theme with a highly structured, dense layout.
   - **Layout/Features:** A seamless 3-panel split. The Left Panel anchors the PM with daily focus items. The Center Panel is a conversational intelligent assistant embedding interactive action cards (e.g., 1-click "Approve Invoice" widgets). The Right Panel dynamically updates data-rich budget and schedule artifacts.

3. **Field Portal (Superintendent/Sub View):**
   - **Role:** The execution interface for the job site.
   - **Aesthetic:** High-contrast dark mode with vibrant neon accents (e.g., green for "Passed"). Built for glare and dusty environments.
   - **Layout/Features:** Mobile-native with massive touch targets. Features a prominent voice dictation button for offline-capable memos and reality-capture photo verification cards to easily sign off on trades' progress.

## 4. The 6 Core Domains
1. **Genesis:** AI takeoffs and automated bid leveling to initiate projects rapidly.
2. **Execution:** Vision-verified progress tracking linked directly to the dynamic CPM-res1.0 scheduling engine.
3. **Network:** Predictive procurement with dynamic vendor scorecards to maintain supply chain resilience.
4. **Ledger:** Smart contracts and automated lien waivers connected to the budget and real-time actuals.
5. **Compliance:** AI-driven permit tracking and OSHA computer vision detection acting as a safety guardrail on site.
6. **Corporate:** Org-wide HR/Payroll (including prevailing wage tracking) and Equipment/Fleet Asset Management (EAM).
