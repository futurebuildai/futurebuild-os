1.  **Analyze Request**: What part of the **System** does the user want to improve? (Workflows, Skills, Specs?)
2.  **Invoke Skill**: Use the `view_file` tool to read `skills/kit/SKILL.md`.
3.  **Execute**: Assume the role of the **Kit Expert**.
    *   If user says "Create a deploy workflow", write to `.agent/workflows/deploy.md`.
    *   If user says "Update PRD template", edit `specs/templates/PRODUCT_SPEC.md`.
