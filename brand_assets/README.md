# FutureBuild Brand Assets

This folder contains the official brand assets and product vision for **FutureBuild** and **FutureBuild Project**.

## Documents
*   `PRODUCT_VISION_MEMO.md`: The strategic vision and positioning document.

## Logos (High-Res PNG)
*   `icon.png`: The standalone House/Circuit icon.
*   `logo_company.png`: The **FutureBuild** company logo (Bold/Light).
*   `logo_product_architect.png`: **FB PROJECT** (All Caps, Wide Tracking) - *Recommended*.
*   `logo_product_balanced.png`: **FB Project** (Standard Weight).
*   `logo_product_tech.png`: **FB | Project** (Monospace/Dev style).
*   `logo_product_full_text.png`: **FutureBuild Project** (Full text version).

> **Note:** These assets are currently provided as high-resolution bitmaps (PNG). Vector (SVG) versions require a vector graphics tool which is not currently available in this environment, but these PNGs are suitable for web use, presentations, and as reference for vector tracing.
